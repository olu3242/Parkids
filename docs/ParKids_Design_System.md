# PAR-KIDS — Design System
**Version:** 1.0

## Brand Personality
Warm | Trustworthy | Family-Friendly | Intelligent | Modern | Encouraging | Calm | Structured

## Color Palette
- Primary Green: #2D7D5A (trust, growth, nature)
- Warm Teal: #3ABFBF (calm, connection)
- Soft Sand/Cream: #FDF6EC (warmth, safety)
- Deep Charcoal: #1E2D2F (text, grounding)
- Accent Amber: #F4A535 (energy, celebration)
- Light Sage: #A8CEAB (backgrounds, cards)

## Typography
- Display: DM Serif Display (warm, editorial headlines)
- Body: DM Sans (clean, legible)
- Child UI: Nunito (rounded, friendly)

## Components
- Buttons: Rounded-xl (12px), solid fill or outlined
- Cards: Rounded-2xl (16px), soft drop shadow
- Inputs: Rounded-xl, 2px border, focus state = green border
- Modals: Rounded-3xl (24px), dimmed backdrop

## Iconography
- Style: Rounded, filled, 2px stroke minimum
- Size: 20px (UI), 24px (action), 32px+ (feature icons)
- Emoji-first for child-facing screens

## Motion
- Duration: 200ms default, 350ms for page transitions
- Easing: ease-out for entrances, ease-in for exits
- Celebration: confetti burst, scale pulse on completion

## Child UI Principles
- Emoji-first selections
- Max 2-3 questions per screen for ages 6-10
- Min touch target: 56x56px
- Grade 3-4 reading level for early age group
- Celebration animations on every completion
